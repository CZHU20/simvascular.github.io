About SimVascular

Description

This project is an open source release of tools for cardiovascular simulation. It includes code for reading 3D images, segmenting structures, generating models and meshes, and modeling blood flow in deformable vessels. Commercial components have been used in the model construction, mesh generation and flow solver components. The SimVascular project is derived from the ASPIRE2 software project and includes modified portions of PHASTA from RPI/SCOREC.

Purpose/Synopsis

Provides a platform for patient-specific cardiovascular modeling and simulation.

Audience

Biomedical researchers interested in Cardiovascular Dynamics, which includes simulations of blood flow as well as vessel dynamics.

Long Term Goals and Related Uses

Allow researchers to go from medical imaging data to hemodynamic (blood flow) simulation results for patient specific models. The application, which integrates custom open-source code and best-in-class commercial components, includes an integrated flow solver with special outflow boundary conditions and fluid-structure interaction capabilities.
