SimVascular TetGen options class. 

TetGenOptions(global_edge_size=None, surface_mesh_flag=None,               
              volume_mesh_flag=None)                                       

----------------------------------------------------------------------     
The TetGenOptions class stores parameter values used to control how        
TetGen generates a finite element mesh.                                    

Example: Create a TetGen options object                                    

    options = sv.meshing.TetGenOptions()                                   

Args: 
  global_edge_size (Optional[float]): The maxium length of an element edge.   
  surface_mesh_flag (Optional[bool]): If True then perform surface remeshing. 
  volume_mesh_flag (Optional[bool]): If True then generate a volume mesh.

 get_values()  
  
  Get the names and values of TetGen mesh generation options.

Data 
allow_multiple_regions
   Type: bool                                                              
   Default: False                                                          
   
   If True then allow for multiple unconnected regions.                    
   


boundary_layer_inside
   Type: bool                                                              
   Default: True                                                           
   
   If True then place the boundary layer inside the solid model surface.   
   This is used for creating a boundary layer for CFD.                     
   


global_edge_size
   Type: float                                                             
   Default: 0.1                                                            
   
   The maxium length of an element edge.            
   


local_edge_size
   Type: list({'face_id':int, 'edge_size':float})                         
   Default: []                                                            
   
   Specify a list of local edge sizes for solid model faces. This is used 
   to create different mesh densities for different regions of the solid  
   model.                                                                 
   
   The 'face_id' parameter is an integer ID identifying a solid model     
   face. The solid model face IDs are obtained using the TetGen mesher    
   get_model_face_ids() method.                                           
   
   The 'edge_size' parameter is the maxium length of an element edge for  
   the face.                                                              
   
   Edge size values are not checked when the values are set. Errors in    
   the values (e.g. a misspelled dict key) are not reported until they    
   used for generating a mesh.                                            
   
   Using local edge sizes for meshing is enabled by setting the           
   'local_edge_size_on' meshing option to True.                           
   
   Example: Setting values for the local edge sizes option                
   
      options.local_edge_size = [ {'face_id':1, 'edge_size':0.3} ]        
      options.local_edge_size_on = True                                   
   


local_edge_size_on
   Type: bool                                                             
   Default: False                                                         
   
   If True then enable using local edge sizes for meshing.                
   


no_bisect
   Type: bool                                                              
   Default: True                                                           
   If True then use the solid model surface mesh for the volume mesh.      
   


no_merge
   Type: bool                                                              
   Default: True                                                           
   
   If True then do not merge coplanar faces.                               
   


optimization
   Type: int                                                               
   Default: 3                                                              
   
   The number of times to optimize the mesh. Optimization moves nodes to   
   obtain a better quality mesh.                                           
   


quality_ratio
   Type: float                                                             
   Default: 1.4                                                            
   
   he quality measure for the mesh. This number corresponds to the ratio   
   between the radius of the circumsphere of an element and the maximum    
   edge size.                                                              
   


radius_meshing_compute_centerlines
   radius_meshing_compute_centerlines

radius_meshing_on
   Type: bool                                                             
   Default: False                                                         
   
   If True then enable radius-based meshing.                              
   


sphere_refinement
   Type: list({'edge_size':float, 'radius':float, 'center':[float, float, float]}) 
   Default: []                                                            
   
   Specify a list of edge size, radius and center for spheres used for    
   sphere refinement meshing. This is used to create different mesh       
   densities for different regions of the solid model based on sphere     
   geometry.                                                              
   
   The 'edge_size' parameter is the maxium length of an element edge for  
   the nodes within a sphere.                                             
   
   The 'radius' parameter is the radius of a sphere.                      
   
   The 'center' parameter is the center of a sphere.                      
   
   Sphere refinement values are not checked when the values are set.      
   Errors in the values (e.g. a misspelled dict key) are not reported     
   until they used for generating a mesh.                                 
   
   Using sphere refinemen for meshing is enabled by setting the           
   'sphere_refinement_on' meshing option to True.                         
   
   Example: Setting values for the sphere refinement option               
   
      options.sphere_refinement.append( {'edge_size':0.2, 'radius':3.7,  
         'center':[1.4, -1.0, 0.078] })                                  
      options.sphere_refinement_on = True                                
   


sphere_refinement_on
   Type: bool                                                             
   Default: False                                                         
   
   If True then enable sphere refinement meshing.                         
   


surface_mesh_flag
   Type: bool                                                              
   Default: True                                                           
   
   If True then perform surface remeshing.                                 
   


use_mmg
   Type: bool                                                              
   Default: True                                                           
   
   If True then use MMG for remeshing.                                     
   


volume_mesh_flag
   Type: bool                                                              
   Default: True                                                           
   
   If True then generate a volume mesh.
